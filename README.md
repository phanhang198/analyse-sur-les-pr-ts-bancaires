# Analyse de la performance financière des ventes de plantes

## Contexte général 📌

Author: Hang NGUYEN  
Tool Used: SQL, PowerBi

L’entreprise dispose d’un fichier de données structuré autour de la vente de produits horticoles/plantes, 
combinant des informations transactionnelles, clients (comptes) et un référentiel produit.
Ces données visent à analyser la performance commerciale, la rentabilité et la répartition géographique des ventes.

Description des sources de données
1. Table Plant_FACT (faits de vente)

Objectifs analytiques pour une Data Analyst

#### ❓ Enjeux métiers :

1 / Identifier les produits et familles les plus rentables

2 / Analyser la contribution des clients et des zones géographiques au chiffre d’affaires

3 / Suivre l’évolution des ventes et des marges dans le temps

4 / Optimiser le portefeuille produit (type, taille, usage)

5 / Aider à la prise de décision commerciale et stratégique

Valeur métier

Ce jeu de données permet de relier performance financière, comportement client et caractéristiques produit. Il constitue une base solide pour la création de tableaux de bord, d’analyses de rentabilité et de recommandations stratégiques orientées business.
