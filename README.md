Analyse de la performance financière des ventes de plantes

📌Contexte général
Author: Hang NGUYEN  
Tool Used: SQL, PowerBi
L’entreprise dispose d’un fichier de données structuré autour de la vente de produits horticoles/plantes, 
combinant des informations transactionnelles, clients (comptes) et un référentiel produit.
